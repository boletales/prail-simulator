あそびかた

  • 全般：
    • 視点の移動：クリック・ドラッグ・ホイール
    • 一時停止・再開：Space キー
    • セーブ・ロード：「一時保存」でブラウザにセーブ、「書き出し」でファイルとしてダウンロード、「参照……」でファイルをロード、「作例読み込み」でプリセットを読み込み。
    • リセット：「作例読み込み」から「空っぽのレイアウト」を選択
    • ※ オンラインモード（/online/*）において、操作は同一のURLに接続しているユーザー間で共有・同期されます。
  • レール操作：
    • 選択：レールの端付近を左クリック。矢印キーで選択点を変更。
    • 追加：W, A, D キーで、選択中の端点から直線・曲線左・曲線右レールを追加。その他のレールのキー割り当てについてはUI右上のタブ（GUI）を参照。
    • 削除：端点を選択して BackSpace または Delete。
    • 向きの変更：F キーで反転、R キーで回転。
    • ポイント切り替え：V キー。
  • 列車操作：
    • ⚠️ 信号機を設置していない場合は「信号に従って走行」のチェックを外さないと走りません！
    • 追加：レールを選択して + キー。
    • 列車選択：列車のいるレールを選択して U キー、もしくは「列車操作」UIで選択。
    • 反転： ; キーで進行方向反転。
    • 削除： * キーで削除。
    • コントロールパネル：
      • 列車番号：列車の上に表示する注釈（編成番号等）を設定できます。
      • 列車タグ：影響を受ける信号ルールの種類の指定です（改行区切りで複数指定可能）。
  • 信号・自動化（信号機操作）：
    • 「信号に従って走行」モードでは、各列車は前方の信号機の現示（色）に従って進行し、ATS-Pに近い動作で減速・停止します。
    • 分岐の手前・奥に信号機を設置すると、I, O キーで進路を構成できます。
    • 前方に分岐・交差がある複雑な信号機は、列車の通過・進路の変更があると赤背景（停止現示）になります。
    • 信号機ルールを設定すると、進路の構成・開通を「列車タグ」に従って自動で行います。
    • 設置・削除：0 で信号機を設置、9 で進入禁止標識を設置、- で削除。
    • 進路の構成：I, O キーで、一つ右・左の進路を構成。
    • 進路の開通：赤背景になった信号を選択して、P キーで開通。
    • 信号機ルール（DSL仕様）：
      • c ：この信号機を「複雑な信号機」として扱います。
      • m [列車タグ] [最高速度] ：該当する列車が通過する際に速度を制限します。
      • o [列車タグ] [進路番号] ：該当する列車が信号機を見つけた際、その方向に進路を開通して自動停止現示を解除します。
      • u [列車タグ] [改変部分] [改変後] ：該当する列車が信号機を見つけた際、列車タグを置換します。
      • s [列車タグ] ：該当する列車を停止させます。
      • O [列車タグ] [進路番号] ：該当する列車を一時停止させ、その後に o と同じように振る舞います。
      • U [列車タグ] [改変部分] [改変後] ：該当する列車を一時停止させ、その後に u と同じように振る舞います。
      • r [列車タグ] ：該当する列車を反転させます。
      • R [列車タグ] [改変部分] [改変後] ：該当する列車を反転させ、その後に u と同じように振る舞います。
      • ※ 進路番号は、右から順に 0, 1, 2... です。
      • ※ 列車タグには正規表現を用いることができます。
      • ※ その他の文字列はコメントとして扱われます。
  • 下側コントロールパネル：
    • 座標表示：現在選択しているレール接続点の座標を表示します。
    • α値表示：直線レールの√2/2倍の長さを合わせるための追加情報です。
    • save / 一時保存：一人用モードであればブラウザへのセーブを試みます。
    • download / 書き出し：レイアウトデータをJSONとしてローカルにダウンロードします。
    • 参照 / ファイル選択：レイアウトデータ（JSON）をアップロードしてロードします。
    • レール注釈：レールの上に駅名などの注釈を表示します。
    • 信号制御：選択中の接続点に信号がある場合、信号ルール（DSL）を設定します。
    • ノッチ操作：信号による自動制限がない場合、指定した手動ノッチで走行します。
    • 列車番号：列車の上に表示する注釈（編成番号等）を設定できます。
    • 列車タグ：列車が影響を受ける信号ルールの種類（列車タグ）を改行区切りで複数指定できます。
